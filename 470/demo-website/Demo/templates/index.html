About

Flask is a free Python web framework for building great Web sites and Web applications using HTML, CSS and JavaScript.

Easy to setup

Run a basic application with only a few lines of code.

Uses Jinja2 templating

Inherit layouts, inject variables, iterate over collections and more all within your HTML.

100% WSGI 1.0 compliant

Adheres to the standard interface between web servers and Python web applications.

Extensively documented

You'll always find what you're looking for.

Connect to any database

Flask provides no native database support, which allows you to choose whatever database works best for your application. There are numerous Flask Extensions available for easily connecting to your favourite DBMS.

Relational

Connect to any SQL database using SQL Alchemy.

Document-based

Connect to your favourite document-based database, such as MongoDB or CouchDB, using Flask PyMongo or Flask CouchDB.

Or anything else

DynamoDB, Cassandra, Cosmos, whatever you like.
